# aiy-wish.github.io

Personal portfolio site for Ayush Chakraborty — Data Engineer @ HabiTerre.

**Live:** [aiy-wish.github.io](https://aiy-wish.github.io)

## Stack

- **Framework:** Next.js 15 (static export)
- **Styling:** Tailwind CSS + CSS custom properties
- **Fonts:** JetBrains Mono + DM Serif Display
- **Charts:** Chart.js (bar + radar)
- **Deployment:** GitHub Pages via Actions

## Features

- Light/dark theme toggle
- Interactive terminal with typewriter animation
- Responsive grid layout
- Component-based architecture (Hero, Projects, Skills, Experience, Education)

## Development

```bash
npm install
npm run dev
```

## Build & Deploy

Deploys automatically to GitHub Pages on push to `main`/`master` via GitHub Actions.

To build locally:

```bash
npm run build
```

Static output goes to `out/`.
